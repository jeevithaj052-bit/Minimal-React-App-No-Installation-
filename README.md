Minimal React App (No Installation)

This project demonstrates how to build a simple React application without installing Node.js or any tools.

🚀 Features

- Uses React via CDN
- No installation required
- Single HTML file
- Beginner-friendly

🛠️ Technologies Used

- React (CDN)
- ReactDOM
- HTML
- JavaScript

📂 Project Structure

index.html

▶️ How to Run

1. Download or clone this repository
2. Open "index.html" in any web browser
3. You will see Hello World!

💡 How It Works

- React is loaded using CDN links
- "React.createElement()" creates UI elements
- "ReactDOM.render()" displays them in the browser

📸 Output

- Displays "Hello World!"
- Hover shows tooltip "Outer div"

🎯 Purpose

This project is useful for beginners to understand:

- Basic React without setup
- Virtual DOM concept
- How React renders elements
